Sobre a ExtraBits

Sobre a empresa

Fundada em 2009, a ExtraBits é especializada em Suporte Técnico, Manutenção, Venda de Computadores de Alta Performance, e tudo o que abrange a informática. Conta com sua Web Store para venda de seus produtos e com sua Loja Física e Assistência Técnica localizada em Novo Hamburgo (RS). O comprometimento e a qualidade no atendimento aos clientes e em seus serviços prestados, está sempre em primeiro lugar.



O que é um ERP?

Enterprise Resource Planning e trata-se de um software integrado de gestão empresarial que reúne numa única solução as informações gerenciais dos setores de uma empresa, tais como: como Contabilidade, Finanças, Fiscal, RH, Suprimentos, Patrimônio e Vendas.

Tecnologias Utilizadas

Front end

HTML 5

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS 3

CSS é um mecanismo para adicionar estilo a um documento web. Em vez de colocar a formatação dentro do documento, o CSS cria um link para uma página que contém os estilos.

Javascript

JavaScript é uma linguagem de programação interpretada. Controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido.

   
Angular

AngularJS é um framework JavaScript código aberto, mantido pelo Google, que auxilia na execução de single-page applications.

Bootstrap 4

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.

Back end

C#

C# é uma linguagem de programação, multiparadigma, de tipagem forte, desenvolvida pela Microsoft como parte da plataforma .NET. A sua sintaxe orientada a objetos foi baseada no C++ mas inclui muitas influências de outras linguagens de programação, como Object Pascal e, principalmente, Java.

Banco de dados SQL

Structured Query Language, é a linguagem de pesquisa declarativa padrão para banco de dados relacional. Muitas das características originais do SQL foram inspiradas na álgebra relacional.

PHP

PHP é uma linguagem interpretada livre, usada originalmente apenas para o desenvolvimento de aplicações presentes e atuantes no lado do servidor, capazes de gerar conteúdo dinâmico na World Wide Web.
